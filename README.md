# 📚 Book Management System

A simple Book Management System built using **Python** and **HTML** to manage library books efficiently.

## Features
- Add books
- Update book details
- Delete books
- Search books
- Issue and return books

## Technologies Used
- Python
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Run `backend.py`.
3. Open `frontend.html` in your browser.
